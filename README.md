## Run locally

Set up and run both of the following services locally:
- https://github.com/kchungradio/newsbody-stories-api
- https://github.com/kchungradio/auth-api

`git clone` this repository.

Install dependencies and run:
```bash
yarn
yarn dev
```
_Or alternatively:_
```bash
npm install
npm run dev
```
